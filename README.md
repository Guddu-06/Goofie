# 🐾 Goofie - Chat App

Goofie is a realtime chat app built with Vite, React, TypeScript, Firebase, and other technologies.

> Built & Designed by [Saswat Rath](https://github.com/Guddu-06)

## 🚀 Features

- 🗝 Sign in with Google
- 👤 View your own profile
- 🗨 Create conversations (1-on-1 or group)
- 📬 Send files, images, and text
- 🔗 Auto-detect links
- 💬 Reply to messages
- 📥 Drag and drop to upload
- 🔔 Unseen message notifications
- 👀 Seen indicators
- 😂 Send reactions to messages
- 😊 Emoji picker
- 👁 View reactions
- 📸 View sent images and files
- 🔄 Change group picture and name
- 🌓 Light / Dark mode
- 🚮 Remove messages
- 👑 Group admin controls

## 🛠️ Tech Stack
- `React` · `Firebase` · `TypeScript` · `Styled Components` · `Vite`

## ⚙️ Setup
1. Clone the repo
2. `npm install`
3. Add your Firebase config to `.env`
4. `npm run dev`

*Enhanced version of Chatify by MariyaTech*
